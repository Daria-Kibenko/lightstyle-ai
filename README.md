# lightstyle-ai

Работа выполнялась в рамках университетской практики.

Название задачи: "Классификация изображений с использованием нейронных сетей".

Условие:
Нейронные сети позволяют решать сложные задачи классификации изображений благодаря их способности выделять особенности объектов. Студентам предлагается разработать модель нейронной сети для автоматической классификации светильников по стилям. Задача включает сбор датасета, обучение модели на примерах и проверку её работы на новых изображениях.
